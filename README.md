# java-database-operations
"A collection of Java Database Connectivity (JDBC) projects demonstrating CRUD operations using MySQL and Eclipse IDE."

## 🛠 Technologies Used
* *Language:* Java
* *Database:* MySQL
* *IDE:* Eclipse
* *Driver:* MySQL Connector/J

## 📂 Projects Included
1. *JdbcInsertDemo:* How to insert records into a MySQL table.
2. *JdbcSelectDemo:* How to fetch and display data.
3. *JdbcUpdate&Delete:* Demonstrating how to modify and remove existing records.
4. *JdbcExample:* General database connection setup.
